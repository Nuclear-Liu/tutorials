> 知识点： `动态规划`


## 描述


王强决定把年终奖用于购物，他把想买的物品分为两类：
**主件**与**附件**，**附件是从属于某个主件**的，下表就是一些主件与附件的例子：


| 主件  | 附件      |
|-----|---------|
| 电脑  | 打印机，扫描仪 |
| 书柜  | 图书      |
| 书桌  | 台灯、文具   |
| 工作椅 | 无       |


如果要买归类为附件的物品，**必须先买该附件所属的主件**，且**每件物品只能购买一次**。
每个主件可以有 **0** 个、 **1** 个或 **2** 个附件。附件不再有从属于自己的附件。
王强查到了每件物品的价格（都是 **10** 元的整数倍），而他只有 **N** 元的预算。
