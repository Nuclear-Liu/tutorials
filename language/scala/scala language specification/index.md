## 前言

Scala 是类 Java 的编程语言，统一了**面向对象**和**函数式**编程；
纯粹的面向对象语言，从某种意义上说，每个值都是一个对象。

Scala 函数式编程：每个函数都是一个价值。
函数定义和高阶函数嵌套是自然支持。
Scala 支持模式的一般概念，可以模拟许多函数中使用的代数类型的匹配语言。

Scala 设计与 Java 无缝互操作。
Scala 类可以调用 Java 方法，创建 Java 对象，从 Java 继承类并实现 Java 接口。
